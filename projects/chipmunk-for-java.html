Chipmunk for Java

Languages
C, Java

Java (JNI) for 2D physics library Chipmunk. Chipmunk is written in C and Chipmunk for Java is a mix of Java and C. Tested and works on Android using the Native Development Kit. Chipmunk for Java is open source and published under the GPLv3.

Video 1: Video of a demo simulation powered by Chipmunk for Java, running on Android Dev Phone 1.

Requirements

  • A JDK.
  • A C Compiler.
